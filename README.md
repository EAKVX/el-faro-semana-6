# 📰 El Faro - Semana 6

## 📌 Descripción del Proyecto
El Faro es un sitio web de noticias desarrollado en PHP utilizando el patrón de arquitectura MVC (Modelo - Vista - Controlador).  
El proyecto permite la interacción con usuarios mediante formularios, incluyendo registro de cuentas y contacto.

---

## 🧩 Tecnologías Utilizadas
- PHP (Programación Orientada a Objetos)
- HTML5
- CSS3
- JavaScript
- Framework Bulma

---

## 🏗️ Arquitectura MVC

El proyecto está estructurado bajo el patrón MVC:

- **Model:** Manejo de datos (Usuario, Articulo)
- **View:** Interfaz del usuario (vistas en PHP)
- **Controller:** Lógica de negocio (procesamiento de formularios)

---

## 📂 Estructura del Proyecto

app/
├── Controller/
├── Model/
├── View/
Config/
Public/
├── CSS/
├── JS/
└── index.php


---

## 🧾 Funcionalidades

- Registro de usuarios
- Formulario de contacto
- Creación de artículos (vista)
- Navegación entre secciones
- Diseño responsive con Bulma

---

## 🌐 Ejecución del Proyecto

Para ejecutar el proyecto en entorno local:

1. Instalar XAMPP :
   https://www.apachefriends.org/es/download.html
   
2. Copiar el proyecto en:
   C:\xampp\htdocs\

3. Iniciar Apache desde XAMPP
   
4. Acceder en el navegador:
   http://localhost/El_Faro_Web_Semana-6/public/

---

## ⚠️ Nota
Este proyecto utiliza PHP, por lo que requiere un servidor local (XAMPP) para su ejecución.
